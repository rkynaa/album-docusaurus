Creates a new archive item.

Authorizations:
bearerAuth
Request Body schema:
required
album
required
string

The unique identifier of the archive.

name
string or null <= 255 characters

The name of the archive item.

item_type
required
string
Enum: "mes" "acc" "med" "doc" "seed" "nft"

The type of the archive item, it can be one of the following:

  • mes - Message
  • acc - Digital Account
  • med - Media
  • doc - Document
  • seed - Seed Phrase
  • nft - NFT
checksum
string or null <= 255 characters

The MD5 checksum of the archive item.

extension
string <= 50 characters

The extension type of the archive item.

file
required
string <uri>

The archive item file.

iv
string or null <= 200 characters

The initialization vector of the archive item for encryption.
